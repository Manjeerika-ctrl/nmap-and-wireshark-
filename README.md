# Task 1 - Open Port Scanning using Nmap

## Objective
To scan the local network and identify open ports and services.

## Tools Used
- Nmap
- Wireshark (optional)

## Commands Used
nmap -sS 192.168.1.0/24

## Findings
- Found multiple devices
- Identified HTTP, HTTPS, and SSH services

## Security Risks
- Open SSH ports may allow brute force attacks
- Telnet is insecure

## Conclusion
Learned basics of network reconnaissance and port scanning.
